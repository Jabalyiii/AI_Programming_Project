# Africa International University Chatbot

A customized university assistant chatbot for Africa International University (AIU).

## Features

- **Intent Recognition** - Intelligently detects user intent from keywords
- **Multiple Programs** - Information about AI & Cybersecurity, IT, Accounting, and Theology programs
- **Campus Information** - Details about library, chapel, and finance office
- **Admissions Support** - Guidance on registration and enrollment
- **Natural Responses** - Multiple response variations for more engaging conversations

## Usage

Run the chatbot:
```bash
python chatbot.py
```

Then type your questions about:
- AI & Cybersecurity program
- Information Technology
- Accounting program
- Theology program
- Campus locations
- Admissions and registration

Type `exit` or `bye` to quit.

## How It Works

The chatbot uses:
- **Keyword matching** - Scores user input against predefined keywords
- **Intent classification** - Groups similar questions together
- **Response randomization** - Picks random responses from categories for variety
- **Conversation history** - Tracks interactions for future enhancement
